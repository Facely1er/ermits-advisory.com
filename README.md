# ERMITS Advisory - Cybersecurity Consulting Platform

A comprehensive cybersecurity consulting platform built with React, TypeScript, and modern web technologies.

## 🚀 Features

- **STEEL Framework**: Enhanced PESTEL analysis with cybersecurity-specific factors
- **RAPID Implementation**: Streamlined cybersecurity implementation methodology
- **Risk Radar**: Real-time threat intelligence and risk assessment
- **Multi-language Support**: English and French localization
- **Dark/Light Theme**: Responsive design with theme switching
- **Interactive Dashboards**: Real-time metrics and visualizations

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + Framer Motion
- **Charts**: Chart.js + React-Chartjs-2
- **Deployment**: Vercel
- **Analytics**: Vercel Analytics

## 📦 Installation

```bash
# Install dependencies
npm install

# Development
npm run dev

# Production build
npm run build:prod

# Preview production build
npm run preview

# Linting and type checking
npm run lint
npm run type-check
```

## 🔧 Environment Configuration

Copy `.env.example` to `.env` and configure your environment variables:

```bash
cp .env.example .env
```
