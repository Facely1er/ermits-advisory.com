# ERMITS Advisory Documentation

This directory contains all reference documentation for the ERMITS Advisory platform, organized by service area.

## 📁 Directory Structure

### `/advisory/`
Strategic and planning documentation for ERMITS Advisory services:
- **ERMITS_Advisory_Documentation_Index.md** - Navigation guide for all advisory documentation
- **ERMITS_Advisory_Offerings_Review_REVISED.md** - Complete strategic analysis and detailed service catalog (70+ pages)
- **ERMITS_Advisory_Quick_Update_Guide_REVISED.md** - Actionable implementation checklist (40+ pages)
- **ERMITS_Advisory_Revision_Summary.md** - Change documentation and rationale (15+ pages)

### `/vciso/`
Virtual CISO service delivery documentation:
- **ERMITS_vCISO_Service_Delivery_Guide.md** - Complete operational framework for vCISO services (54 pages)
- **vCISO_Implementation_Quick_Start.md** - Quick start guide for delivering vCISO services

### `/steel/`
STEEL Assessment Platform documentation:
- **STEEL_Implementation_Summary.md** - Implementation guide and summary
- **STEEL_Delivery_Package_Summary.md** - Delivery package overview
- **STEEL_Complete_Question_Set.md** - Complete question set with scoring rubrics
- **STEEL_GitHub_Integration_Guide.md** - GitHub integration instructions
- **STEEL_Enhancement_Summary.md** - Enhancement documentation
- **STEEL_Official_Description_FINAL.md** - Official platform description
- **STEEL_Enhanced_Diagrams.md** - Visual diagrams and enhancements
- **STEEL_Assessment_Complete.html** - Complete assessment HTML reference
- **STEEL_Assessment_Platform_Complete.html** - Platform HTML reference
- **STEEL_Enhanced.html** - Enhanced version HTML reference

### `/templates/`
Service delivery templates:
- **Template_Client_Kickoff_Agenda.md** - Client kickoff meeting agenda template (22 pages)
- **Template_Monthly_Executive_Briefing.md** - Monthly executive briefing template (14 pages)

### Root Files
- **Quick_Deploy_Commands.md** - Quick deployment commands and scripts

## 🎯 Usage

These files are **reference documentation** for:
- Developers building the platform
- Content creators updating website content
- Service delivery teams implementing vCISO services
- Strategic planning and service design

## 📝 Notes

- These files are **not imported** into the application code
- They are **reference materials** for development and content creation
- Files can be accessed directly or used as documentation
- Markdown files can be viewed in any markdown viewer
- HTML files are reference implementations
